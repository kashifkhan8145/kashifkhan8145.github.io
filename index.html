Physics-Informed ML for MPPT under Partial Shading

Built an end-to-end ML pipeline (ANN, XGBoost, LightGBM, CatBoost, SVR, GPR, RF, kNN) for PV MPPT. Generated a large synthetic dataset via a physics-based simulator, engineered robust features, and performed grid-search hyperparameter tuning. Evaluated with regression + time-series tracking metrics, inference latency, and Wilcoxon significance tests.

Python Pandas/NumPy/Scikit-learn Synthetic Data Feature Engineering Model Selection Grid Search Statistical Testing Time-Series Evaluation Reproducible Reports

Remaining Useful Life (RUL) — NASA C-MAPSS

Built a ConvFormer-style Transformer for turbofan RUL with leakage-proof evaluation: sliding-window sequences, train-only standardization, lightweight op×sensor interaction features, and GroupKFold cross-validation by unit. Trained with mixed precision on GPU (TF 2.10) using early stopping and ReduceLROnPlateau. Beyond regression metrics, we add bootstrap CIs, EVS over the last 30 cycles, a dense alarm-threshold grid (H,K) with F1/cost/lead-time, and a model-complexity analysis (parameters, estimated FLOPs, per-sample latency).

Highlights (cap=100): FD001 RMSE ≈ 12.8, FD003 RMSE ≈ 10.0; complexity ~1.46M params / 119M est. FLOPs / ~36 ms (FD001 best config).

Transformers TensorFlow (GPU, AMP) GroupKFold CV Alarm Analysis (H,K) Bootstrap CIs Model Complexity Time-Series

Extrapolation-Aware ML Based Behavioral Modeling for Advanced RF Devices (GaN HEMTs)

Built small-signal S-parameter models for a GaN HEMT across wide conditions such as bias conditions, frequencies and temperatures using XGBoost and Grey Wolf Optimizer (GWO) for hyperparameter search. Ran controlled splits to evaluate interpolation vs. extrapolation along temperature, bias, and frequency. Results: temperature extrapolation was robust; frequency was intermediate; bias was most challenging. GWO-XGBoost consistently reduced MSE/MAE and improved R² over the default model. Validated with Smith charts and 3D response maps at representative bands.

XGBoost Grey Wolf Optimizer (GWO) Metaheuristic Tuning Interpolation & Extrapolation RF/mmWave Modeling S-parameters Smith Charts 3D Visualization Python NumPy/Pandas scikit-learn

ML-Assisted Microstrip Filter Design (GWO-ANN)

End-to-end EM → ML surrogate → metaheuristic optimization → fabrication pipeline in Advanced Design System (ADS). Designed and validated a high-frequency microstrip filter using an ANN surrogate and Grey Wolf Optimizer for rapid design-space exploration. Hardware verified with Vector Network Analyzer (VNA) measurements.
